What is the graham number calculator?
The Graham number calculator is a tool that determines a stock's fair value.
How to calculate graham number?
To calculate graham number you should use the following formula:
graham number = (22.5 * earnings per share * book value per share)^0.5
Why use graham number calculator?
The Graham number calculator is used in order to measure a stock's fair value.
